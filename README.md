# Кінопокази просто неба

React додаток для перегляду та реєстрації на кінопокази просто неба.

## Технології

- React 19 + Vite
- Redux Toolkit 
- React Router
- Zod
- Recharts

## Запуск

npm install
npm run dev

## Сторінки

- `/` — список подій з пошуком
- `/participants/:id` — учасники події
- `/register/:id` — реєстрація на подію
- `/analytics` — графіки реєстрацій

## Функціонал

- Перегляд подій з фільтрацією по назві
- Позначення подій як "Цікаво" (зберігається в localStorage)
- Реєстрація учасників з валідацією (Zod): ПІБ, email, дата народження 18+, джерело
- Сторінка аналітики з двома графіками: активність по датах і розподіл по подіях
- Перемикач темної/світлої теми (зберігається в localStorage)
- Адаптивний дизайн
